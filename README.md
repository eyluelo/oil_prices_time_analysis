# oil_prices_time_analysis
Time-analyse of the oil prices and correlating them to the big events after 2000s.

## Overview
After my first project with the Titanic dataset, I wanted to try something different. I used the Brent Oil Prices dataset to see how global events like wars and financial crises actually change the numbers on a graph. This was a great way for me to practice Time-Series data and discover what I can do with Pandas.
## Tools & Technologies

   * Python: My main tool for this analysis.

   * Pandas: I used this to clean the data and manage the dates.

   * Matplotlib & Seaborn: I used these to turn boring numbers into a visual story with lines and markers.

## What I Discovered
   * Change doesn't mean directly going up, sometimes big crises can cause a sudden drop. 

   * The Russia-Ukraine Surprise: I found it very interesting that after the initial spike in the Russia-Ukraine war, prices actually started to go down. It showed me that the global market eventually finds new ways to adapt (like selling to different countries).

   * Learning Visualization: This project helped me learn how to add "event lines" (vertical markers) to a chart to show exactly when a war or a crisis happened.

## Summary
While the Titanic project was about classification, this project taught me how to look at data over time.
